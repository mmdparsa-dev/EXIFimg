# EXIFimg
EXIFimg یک ابزار مینیمال و امن برای مشاهده اطلاعات EXIF و متادیتای تصاویر هست. این سرویس بدون آپلود فایل کار می‌کنه؛ یعنی تمام پردازش‌ها روی دستگاه شما انجام می‌شه (حفظ حریم خصوصی). با EXIFimg می‌تونین تاریخ ثبت تصویر، مدل دوربین، تنظیمات شاتر و دیافراگم، اطلاعات لنز، مختصات GPS (در صورت وجود) و هش فایل را به‌سرعت ببینین.
