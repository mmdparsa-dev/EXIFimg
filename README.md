Overview ✨

Exifimg is an open-source tool for viewing and analyzing image metadata, with a strong focus on EXIF data. It helps you inspect the hidden information stored inside photos and image files, such as camera model, device details, capture time, GPS coordinates, orientation, software tags, and many other metadata fields.

The goal is simple: make metadata inspection fast, clean, and easy to understand without turning it into a bulky forensic suite. Exifimg is designed for people who want a focused tool with a modern interface and a lightweight workflow.

Whether you are a photographer checking technical details, a developer testing metadata parsing, a privacy-conscious user reviewing what a photo reveals, or someone doing lightweight digital investigation, Exifimg makes the process straightforward.

معرفی ✨

Exifimg یک ابزار متن‌باز برای مشاهده و تحلیل متادیتای تصاویر است که تمرکز اصلی آن روی داده‌های EXIF قرار دارد. این پروژه کمک می‌کند اطلاعات پنهان داخل فایل‌های تصویری را ببینید؛ اطلاعاتی مانند مدل دوربین، مشخصات دستگاه، زمان ثبت تصویر، مختصات GPS، جهت تصویر، تگ‌های نرم‌افزاری و سایر فیلدهای متادیتا.

هدف پروژه ساده است: نمایش متادیتا به شکلی سریع، تمیز و قابل فهم، بدون اینکه به یک ابزار سنگین و پیچیده تبدیل شود. Exifimg برای کسانی ساخته شده که یک ابزار متمرکز، مدرن و سبک می‌خواهند.

چه عکاس باشید، چه توسعه‌دهنده، چه کاربر حساس به حریم خصوصی، یا کسی که فقط می‌خواهد بداند یک عکس چه اطلاعاتی را در خود پنهان کرده، Exifimg کار را ساده می‌کند.
